#### 实现中英文简繁转换

```css
/* 可以让网站中简体中文变为繁体*/
body {
    font-variant-east-asian: traditional;
}
```

#### 支持条件

要实现这种效果需要字体本身包含繁体变体。

Windows 系统中的几个默认字体都没有包含繁体变体， OS X，也就是 mac、iphone、ipad 等设备中的默认字体是包含繁体变体的。

因为本文中提到的属性只是在 OS X 和 ios 系统中有效
